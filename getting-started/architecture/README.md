# Architecture Overview

RKPulse is built as a cross-platform ecosystem using Flutter, Next.js, Firebase, and Node.js.

## Key Layers
- **Frontend:** Flutter (mobile), Next.js (web)
- **Backend:** Node.js + Prisma + PostgreSQL
- **Realtime Sync:** Firebase (Auth, Firestore, Crashlytics)

⚠️ Proprietary earning logic is not included in this public documentation.
